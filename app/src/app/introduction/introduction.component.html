Welcome to Bespoke

A Comprehensive .NET Library Suite

Bespoke is a collection of modular, extensible .NET libraries designed to accelerate application development while maintaining high quality and performance standards.

Core Libraries

The foundation library providing essential models, interfaces, and utilities that all other Bespoke libraries build upon.

Features include:

  • Base entity models with standardized properties
  • Common interfaces for consistent implementation patterns
  • Utility classes for everyday development tasks

A flexible logging framework that simplifies application logging with multiple provider options.

Features include:

  • Pluggable logging providers
  • Structured logging support
  • Log level filtering and configuration

A messaging library for implementing various communication patterns in your applications.

Features include:

  • Message models for different communication scenarios
  • Message handlers and processors
  • Support for various messaging patterns

Getting Started

To start using Bespoke libraries in your project, install the NuGet packages: 

dotnet add package Bespoke.Core
dotnet add package Bespoke.Logging
dotnet add package Bespoke.Messaging

Check out our documentation for detailed guides and API references.
